# helohavoc site

Starter structure for helohavoc pages.

- `/` — homepage
- `/mealplan/` — meal planning
- `/dashboard/` — dashboard
- `/tools/` — tools launcher
- `/rc/` — RC workspace
- ## Build notes
- Static Cloudflare Pages site
- Simple HTML starter pages
- RC page stores entries in browser localStorage
